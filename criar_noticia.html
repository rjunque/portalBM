Gerar Código de Notícia

Instruções:

  1. Preencha os campos abaixo com as informações da nova notícia.
  2. Clique em "Gerar Código HTML".
  3. Copie o código gerado na caixa de texto.
  4. Abra o arquivo HTML da categoria correspondente (ex: politica_nacional.html) no seu editor de código.
  5. Cole o código copiado **imediatamente após a tag <h2>** daquela categoria. Isso garantirá que a nova notícia apareça no topo.
  6. Salve o arquivo HTML e faça um novo commit no GitHub para publicar a alteração.
  7. Se a notícia tiver uma imagem, você precisará adicionar a tag <img> manualmente no código gerado.
